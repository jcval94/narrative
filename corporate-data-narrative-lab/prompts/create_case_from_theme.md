# Create Case From Theme

Usa este repositorio para crear un caso nuevo sobre el tema:

```text
[tema]
```

Empieza con `00_curriculum_mapper` y sigue `CODEX_WORKFLOW.md`. Elige el nivel analítico mínimo suficiente usando `DATA_SCIENCE_CURRICULUM.md`. Después diseña el `Narrative Learning Experience Standard`: intriga inicial, rol del alumno, misterio central, historia oficial, historia oculta, pista visual, pregunta activa, traducción de jerga, giro, decisión débil, decisión robusta, cierre tragicómico y regla transferible.

No crees dashboard. Entrega:

- salida de `00_curriculum_mapper`;
- nivel analítico declarado;
- técnica mínima suficiente;
- caso markdown;
- especificación de datos sintéticos;
- especificación visual;
- HTML autocontenido con una gráfica central;
- humor tragicómico integrado;
- validación con `14_quality_gatekeeper`;
- validación con `tools/validate_case_structure.py`.

Reglas: no abrir con contexto breve; no filtros, no pestañas, no KPI cards de monitoreo, no self-service, no explorador de datos. Las tarjetas narrativas solo se permiten si enseñan pista, contraste, cálculo o regla.
