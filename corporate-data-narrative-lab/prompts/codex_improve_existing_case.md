# Mejorar un caso

Conserva el concepto analítico. Reescribe toda línea que suene redactada,
genérica o desconectada de la anterior. Da contexto a números, cambia jerga por
lenguaje hablado, elimina muletillas y preguntas A/B/C, y deja una sola gráfica
y una pausa. Regenera el HTML desde Markdown y valida.
