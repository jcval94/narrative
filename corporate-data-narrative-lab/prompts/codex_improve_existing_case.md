# Codex Improve Existing Case

Mejora el caso:

```text
[ruta del caso]
```

Prioriza en este orden:

1. intriga inicial;
2. rol del alumno o protagonista;
3. tesis;
4. nivel analítico y técnica mínima suficiente;
5. evidencia visual como escena del crimen;
6. conflicto;
7. pregunta activa y reducción de jerga;
8. mala decisión;
9. piloto;
10. cierre tragicómico, regla transferible y pregunta de transferencia;
11. riesgos;
12. humor integrado;
13. estilo.

No agregues complejidad visual innecesaria.

No conviertas el caso en reporte expositivo. Si abre con contexto breve, reescribe la entrada como escena, contradicción, anomalía o pregunta incómoda.

Si el caso no declara nivel, vuelve a `00_curriculum_mapper`. Si parece dashboard o usa KPI cards de monitoreo, devuelve `FAIL` con `14_quality_gatekeeper`.
