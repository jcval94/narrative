# Create Case From Chart

Tengo una gráfica o patrón visual:

```text
[descripción de la gráfica]
```

Construye una narrativa corporativa donde esa visualización funcione como evidencia. Define tesis, conflicto, mala decisión, acción robusta, acción débil, piloto, riesgos y aprendizajes. No conviertas la gráfica en dashboard.

Empieza con `00_curriculum_mapper`, declara nivel analítico y técnica mínima suficiente. Usa HTML simple, vertical y autocontenido. Incluye humor tragicómico controlado y termina con `14_quality_gatekeeper`.

Reglas: no filtros, no pestañas, no KPI cards, no self-service, no explorador de datos.
