# Checklist

- [ ] ¿Hay objetivo de aprendizaje?
- [ ] ¿Hay conceptos enseñados?
- [ ] ¿Hay dificultad?
- [ ] ¿Las preguntas usan la gráfica?
- [ ] ¿La respuesta esperada es accionable?
- [ ] ¿Hay errores comunes?
- [ ] ¿Hay actividad en clase?
- [ ] ¿Hay actividad de práctica?
- [ ] ¿Hay rúbrica breve?
- [ ] ¿La pregunta obliga a conectar técnica mínima suficiente, lectura visual, riesgo de negocio y comunicación con stakeholders?
- [ ] ¿La respuesta esperada distingue criterio robusto de solución superficial?
