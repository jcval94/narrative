# Checklist

- [ ] ¿El caso empieza con intriga antes que contexto?
- [ ] ¿Hay `opening_hook`?
- [ ] ¿Hay `protagonist_or_student_role`?
- [ ] ¿Hay `central_mystery`?
- [ ] ¿Data mira con cuidado?
- [ ] ¿La gráfica revela algo incómodo?
- [ ] ¿Hay `visual_clue` y `student_question`?
- [ ] ¿Se distingue `official_story` de `hidden_story`?
- [ ] ¿La jerga clave está traducida?
- [ ] ¿Negocio propone una salida cómoda?
- [ ] ¿Data advierte un riesgo concreto?
- [ ] ¿La decisión imperfecta es plausible?
- [ ] ¿Hay giro después de algunos meses?
- [ ] ¿Hay cierre tragicómico?
- [ ] ¿Hay `transferable_rule`?
- [ ] ¿El aprendizaje es transferible?
