# Examples

## Ejemplos de tesis fuertes

- Una regla de negocio mal diseñada no solo mide comportamiento: lo crea.
- Recortar datos para ajustarlos a una plataforma vuelve invisible la evidencia.
- Un KPI puede mejorar porque cambió la definición, no porque mejoró el proceso.
- Una etiqueta operativa pobre puede contaminar cualquier modelo posterior.

## Ejemplo débil

"El caso enseña a usar histogramas."

Problema: describe una herramienta, no una verdad de negocio.
