# Checklist

- [ ] ¿La tesis es una frase fuerte?
- [ ] ¿Hay lectura superficial?
- [ ] ¿Hay lectura correcta?
- [ ] ¿Hay `central_mystery`?
- [ ] ¿Hay `official_story` y `hidden_story`?
- [ ] ¿Hay `opening_hook` preliminar sin revelar todo?
- [ ] ¿La tesis evita tecnicismo innecesario?
- [ ] ¿La tesis es incómoda pero plausible?
- [ ] ¿El aprendizaje central está definido?
- [ ] ¿Hay `transferable_rule` breve y accionable?
- [ ] ¿Los límites del caso están claros?
