# Checklist

- [ ] ¿Hay máximo 4 frases de humor?
- [ ] ¿El humor aparece después de decisión, giro o cierre?
- [ ] ¿No interrumpe explicaciones técnicas?
- [ ] ¿No ridiculiza personas?
- [ ] ¿No reduce la seriedad de riesgos éticos?
- [ ] ¿El caso seguiría funcionando sin el humor?
- [ ] ¿La proporción se mantiene cerca de 70% análisis, 20% narrativa y 10% humor?
