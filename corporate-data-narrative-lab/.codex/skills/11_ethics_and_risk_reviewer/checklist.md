# Checklist

- [ ] ¿Los datos son sintéticos?
- [ ] ¿Hay riesgo de vigilancia laboral excesiva?
- [ ] ¿Hay falsos positivos?
- [ ] ¿Hay falsos negativos?
- [ ] ¿La señal es para investigar o accionar?
- [ ] ¿Hay revisión humana?
- [ ] ¿Hay daño posible por automatización?
- [ ] ¿Puede afectar injustamente a un grupo?
- [ ] ¿La narrativa evita datos sensibles?
- [ ] ¿No hay humor en riesgos éticos?
