# Checklist

- [ ] ¿La señal permite una lectura cómoda inicial?
- [ ] ¿Un detalle concreto abre la grieta?
- [ ] ¿La pregunta depende de mirar la imagen?
- [ ] ¿La comparación mínima está clara?
- [ ] ¿La conclusión peligrosa es plausible?
- [ ] ¿La evidencia cambia la decisión?
- [ ] ¿El título evita revelar el giro?
- [ ] ¿La señal visual puede rastrearse a un incentivo, restricción o atajo?
- [ ] ¿La anotación expone una decisión del sistema sin acusar a una persona?
