# Examples

## Ejemplo bueno

Histograma con barras grises para movimientos normales y barras rojas para un pico cercano a -1 peso. Una anotación dice: "salidas pequeñas concentradas cerca del umbral de activación".

## Ejemplo malo

Pantalla con filtros por empleado, sucursal, fecha, producto, KPI cards y tabla descargable.

Problema: ya no es narrativa con evidencia; es dashboard.

## Ejemplo de detención

Si el autor pide agregar filtros para encontrar la señal, detente y vuelve a `03_data_signal_designer`: la evidencia todavía no está lo bastante enfocada.
