# Anti-Patterns

- "Ya que estamos, agreguemos un selector."
- Dashboard disfrazado de caso.
- Filtros.
- Cards.
- Drilldowns.
- Tabs.
- Tablas exploratorias.
- Comparadores complejos.
- Layouts de BI.
- Gráficas decorativas.
- Visualizaciones demasiado complejas para la tesis del caso.
- Leyendas extensas que explican toda la historia.
- Colores decorativos sin función.
- Gráfica técnicamente correcta pero narrativamente muda.
