# 13 HTML Story Renderer

## Funcion

Convierte Markdown canonico en HTML lineal sin agregar, quitar ni reordenar
contenido.

## Contrato

- HTML autocontenido y responsive.
- Sin JavaScript ni recursos externos.
- Titulo, escenas, dialogo, SVG, pausa, explicacion, cierre y regla.
- Personajes visibles en cada intervencion.
- Exactamente una grafica.
- Sin tarjetas, dashboard, navegacion ni layouts alternativos.
