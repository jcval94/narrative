# Repo Map

## Raíz

- `ABSURD_OFFICE_COMEDY_DATA_STANDARD.md`: único estándar narrativo.
- `DATA_SCIENCE_CURRICULUM.md`: conceptos y técnica mínima.
- `NARRATIVE_PRINCIPLES.md`, `HUMOR_GUIDE.md`: escritura y ritmo.
- `DATA_VISUAL_EVIDENCE_GUIDE.md`: gráfica central.
- `CASE_RUBRIC.md`, `CODEX_WORKFLOW.md`: revisión y proceso.

## `.codex/`

Skills locales para concepto, conflicto, evidencia, escritura, humor,
aprendizaje, render y calidad.

## `templates/` y `prompts/`

Contratos cortos y solicitudes reutilizables.

## `examples/`

Dieciocho casos con Markdown canónico, data specs, visual specs y HTML lineal.

## `tools/`

Renderer Markdown a HTML y validadores de historias y HTML.
