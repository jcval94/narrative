# Operating Principles

1. `ABSURD_OFFICE_COMEDY_DATA_STANDARD.md` es el unico estandar narrativo.
2. Markdown es la fuente canonica y HTML una vista lineal.
3. Naturalidad, humor y claridad de datos pesan mas que completar campos.
4. Cada caso usa un story spine corto, una grafica y una pausa educativa.
5. Los ejemplos deben variar personajes, problemas, graficas y remates.
6. La critica apunta al sistema; la persona bajo presion conserva dignidad.
7. Toda afirmacion analitica debe estar sostenida por datos sinteticos
   consistentes y una visualizacion legible.
